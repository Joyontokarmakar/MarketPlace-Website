### **Clone From**:
```
git clone https://github.com/Joyontokarmakar/MarketPlace-Website.git
```
### To Run Project follow the following steps:
## 1.
```
yarn
```
or
```
npm install
```
##2. 
```
yarn dev -o
```
or
```
npm run dev
```
